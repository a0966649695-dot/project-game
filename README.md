# 記憶翻牌小遊戲（Memory Card Game）

## 專題簡介
本專題為一款互動式記憶翻牌小遊戲，玩家需透過翻牌方式找出相同圖案的卡片配對。
專題開發過程中使用 Git 與 GitHub 進行版本控制與協作，並設計多種特色玩法以提升遊戲的挑戰性與趣味性。

## 使用工具與技術
- HTML、CSS、JavaScript
- Visual Studio Code
- Git、GitHub
- ChatGPT（輔助程式撰寫與說明）

## 使用技術
- HTML

## 遊戲玩法
1. 遊戲開始後，畫面會顯示多張背面卡片
2. 玩家每次可翻開兩張卡片
3. 若圖案相同則配對成功，否則卡片會翻回
4. 在時間與嘗試次數限制內完成所有配對即可獲勝

## 特色玩法設計
1. **限時模式**：玩家需在倒數時間內完成遊戲
2. **難度選擇**：可選擇不同卡片數量（簡單 / 困難）
3. **嘗試次數限制**：限制翻牌次數，增加策略性
